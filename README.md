# Slurm-Job-Monitor
Slurm job monitor. You can open a pseudo-graphical interface on the command line to monitor the Slurm status in real time
## Usage
| key  | Description  |
| ---  | -----------  |
| h  | help page  |
| J  | job page  |
| N  | node page  |
| S  | information of all node  |
| F  | find a single node information  |
| j  | next row  |
| k  | previous row  |
| q  | quit  |
